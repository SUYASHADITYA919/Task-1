My Quizzo 🧠
My Quizzo is a sleek, responsive, and interactive quizzing application designed to provide a seamless user experience.

🚀 Features:
Dynamic UI: A clean, mobile-first design powered by Tailwind CSS.

Score Calculation: Automated scoring logic to track performance.

Navigation System: Intuitive flow between questions and the final results page.

⌨️ Tech Stack:
HTML5: Semantic structure for better accessibility.

Tailwind CSS: Utility-first styling for a custom, modern look.

JavaScript (ES6+): DOM manipulation, state management, and logic.
